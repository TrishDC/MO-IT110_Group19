# MO-IT110 Object-Oriented Programming | Group 19

##  Project Overview
This repository contains the source code and files for our **MO-IT110** group project.  
It is built using **Java** (NetBeans) and demonstrates core concepts such as:
- OOP (Classes, Inheritance, Polymorphism)
- Basic program structure and organization

##  Tools Used
- Java 
- Apache NetBeans
- GitHub for version control
